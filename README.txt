	fempkg is a Finite Elements package used to study various transport 
phenomena problems, mainly in fluid dynamics. It's development started as part 
of the author's masters thesis (ongoing) 

Requirements:
	-SciPy       (https://www.scipy.org/)
	-MeshIO      (https://github.com/nschloe/meshio)
	-Gmsh        (https://gmsh.info/)
	-Matplotlib  (https://matplotlib.org/)



Subpackages:

	- mesh   

		Contains mesh reader and mesh classes and methods	

	- fem

		Matrices Assembly, Boundary Conditions, Gaussian Quadrature
		Types of Elements			

	- ale

		Arbitrary Lagrangian Eulerian Method (ALE)

	- sl

		Semi-Lagrangian Method

	- filesys

		Folder organization and input/output of results, files, etc.

	- utils

		Various utilities modules, functions and methods.
